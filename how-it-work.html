Smart Home Control Panel

How It Works

1. ESP32 Microcontroller

The ESP32 connects to your Wi-Fi and lighting circuit. It constantly checks Firebase for status changes and controls the lights accordingly.

2. Firebase Realtime Database

Firebase syncs the status of the lights. Any button press from the website instantly updates the value in the database.

3. Website Interface

Accessible from mobile or laptop. Users press On/Off buttons to change light status—no app installation needed.

4. Real-Time Operation

The ESP32 responds to any Firebase update in real time, ensuring the lights react quickly and reliably.

No sensors or physical switches—just tap a button and control your home lights anywhere with internet access.
